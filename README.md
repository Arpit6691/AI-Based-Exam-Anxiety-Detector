# AI-Based Exam Anxiety Detector

An **AI-powered mental wellness support system** that analyzes student text inputs to detect exam-related anxiety levels using **Natural Language Processing (NLP)** and a **BERT-based deep learning model**.

The system classifies anxiety into **Low, Moderate, or High levels** and provides supportive feedback and calming tips to help students manage exam stress.

---

## Project Overview

Students often experience stress and nervousness before examinations but may find it difficult to express their feelings openly. The **AI-Based Exam Anxiety Detector** allows students to enter their thoughts or reflections, which are then analyzed by an AI model to determine their anxiety level.

The system performs **real-time text analysis** and provides helpful insights while maintaining **privacy and ethical AI practices**.

---

## Features

* AI-powered **exam anxiety detection**
* **BERT-based NLP model** for text analysis
* Classification into **Low, Moderate, and High anxiety**
* **Real-time feedback**
* Simple and interactive **Streamlit user interface**
* **FastAPI backend** for model inference
* Privacy-focused and **non-diagnostic support tool**

---

## Tech Stack

* **Python**
* **Natural Language Processing (NLP)**

* **FastAPI**

---

## System Architecture

1. **User Input**
   Students enter their thoughts or feelings about upcoming exams.

2. **Text Processing**
   The system preprocesses the text using NLP techniques.

3. **AI Model Prediction**
   A trained **BERT model** analyzes the text and predicts anxiety levels.

4. **Result Display**
   The system shows the predicted anxiety category with supportive suggestions.

---

## Use Case Scenarios

### 1. Pre-Exam Anxiety Assessment

Students can enter their thoughts before exams, and the system analyzes their emotional state to provide feedback and calming tips.

### 2. Educational Institution Monitoring

Educational institutions can analyze anonymized anxiety trends among students before major exams and provide counseling or support programs when needed.

---

## Ethical Considerations

This system is designed as a **supportive tool only** and **not as a medical diagnostic system**.

* User inputs remain **anonymous**
* Data privacy is maintained
* Results are meant for **awareness and support**
